# 💎 Integración con Obsidian

Una de las mayores ventajas de **VanillaDocs** es que todo su contenido está escrito en Markdown estándar y guardado en archivos físicos dentro de la carpeta `docs/`.

Esto significa que **no estás atado a un panel de administración web lento** para escribir tu temario.

Puedes usar **Obsidian**, el editor de Markdown más potente del mundo, para escribir y organizar todas tus clases a velocidad de la luz.

## ¿Cómo hacerlo?

1. Instala [Obsidian](https://obsidian.md/) (es 100% gratuito).
2. Haz clic en **"Open folder as vault"** (Abrir carpeta como bóveda).
3. Selecciona la carpeta `docs/` de tu proyecto VanillaDocs.

¡Listo! A partir de ese momento podrás:
* Usar el atajo `Ctrl+O` para saltar entre temas al instante.
* Usar la vista de gráfico para ver cómo se conectan tus lecciones.
* Escribir Markdown súper rápido con la ayuda visual de Obsidian.
* Renombrar archivos y que los enlaces se actualicen solos.

Como todo es local, en cuanto guardes tu archivo en Obsidian, simplemente refresca tu VanillaDocs en el navegador y verás los cambios reflejados al instante.
